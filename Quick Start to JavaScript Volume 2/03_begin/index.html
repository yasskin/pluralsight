Loops

Loops are another form of control flow. They "loop" through a block of code until an original specified condition is no longer true.

For loops are an incredibly useful kind of loop. Always make sure your loop eventually evaluates to false or you will create what is called an infinite loop and crash the browser.

View syntax example

                //prints numbers 1-10 to the console
                
                for( i = 0; i < 11; i++){
                  console.log(i);
                }
