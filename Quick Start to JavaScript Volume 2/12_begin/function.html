Functions

What if we wanted to do the same set of steps multiple times? It would be very tiring to type out the same thing over and over again, and it would make our code very messy. Functions allow us to associate a command with an action.

We need the following to create a function:

  • function keyword
  • A name for the function
  • (Optional) Any necessary parameters, comma separated in parenthesis
  • Code block to be executed when function is called

When you call a function, the information you pass to the function (the part in parenthesis) is called an argument.

View syntax example

                
                function sayHello(name) {
                  console.log("Hello " + name);
                }
                
                sayHello("Mark");
