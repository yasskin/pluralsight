Switching Things Up

Another way of using control flow, or writing code in a non-linear fashion, is by using a switch statement. This is similar to how an if/else statement checks for multiple "if" conditions, and runs code only if those conditions have been met.

View syntax example

switch(n) {
case example1:
    console.log("This code will run if n === example1 is true.");
    break;
case example2:
    console.log("This code will run if n === example2 is true.");
    break;
case example3:
    console.log("This code will run if n === example3 is true.");
    break;
default:
    console.log("Default will run if no cases are met.");
    break;
}
